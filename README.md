# AgriTrack-Farm-Management
AgriTrack is a web-based farm management application designed to help farmers and farm owners manage agricultural activities efficiently. It provides an easy-to-use platform for monitoring crops, tracking resources, and organizing important farm-related information.

## Features
- User-friendly and responsive interface
- Farm activity management
- Crop and resource tracking
- Secure backend integration
- Real-time data handling
- Organized and scalable project structure

## Technologies Used
### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
### Backend
- Node.js
- Express.js
### Database
- MongoDB

## Prerequisites
Make sure you have the following installed:
- Node.js
- npm (Node Package Manager)
- MongoDB

## Running the Application
### Start the Backend Server
cd backend
node server.js
### Start the Frontend Application
cd frontend
npm start

The frontend will run on http://localhost:3000.
